An interactive, publicly accessible map of residential, commercial, and investment property transfers recorded with the La Crosse County Register of Deeds. Built for LADCO to track economic development activity across the region.

What This Is

This map displays Real Estate Transfer Return (RETR) data for property transfers in La Crosse County, including commercial, industrial, agricultural, vacant land, mixed-use, and multi-family investment properties. Data is sourced from the Wisconsin Register of Deeds RETR system and updated monthly by LADCO staff.

The public can use the map to explore transfer activity by location, filter by property type, date range, municipality, conveyance type, and sale price, and distinguish between arm's-length market sales and non-arm's-length transfers such as LLC reorganizations, trust conveyances, and gifts.


Features


Interactive Google Maps base with color-coded pins by property type
Pin size scaled to sale price — larger pins indicate higher-value transactions
Arm's-length sales displayed at full opacity; non-arm's-length transfers at reduced opacity
Toggle to show arm's-length sales only
Filters for date range, municipality, property type, document type, conveyance type, sale price range, and number of units
Text search across address, grantor name, grantee name, and parcel number
Sidebar list sorted by most recent recorded date
Breakdown tab with bar charts by property type, municipality, conveyance type, and year
Stats bar showing total transfers, filtered count, arm's-length sales count, total sales volume, and median sale price
Parcel number links to La Crosse County GIS viewer
CSV export of currently filtered results
Fully responsive — works on desktop and mobile
